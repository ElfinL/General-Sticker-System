⚙️ 初始設定

此版本使用 GitHub Gist 作為免費資料庫，請先建立一個 Gist：

  1. 前往 gist.github.com
  2. 建立一個新的 Gist，檔名填 GSS-signatures.json
  3. 內容先填入 [] (空陣列)
  4. 點擊 "Create secret gist" (建議用 secret 避免被搜尋)
  5. 複製 Gist ID (網址中最後那串字符)
